## 现在支持的 Markdown 格式
- 标题（1~6级）（见上）
- 强调（**加粗**、_斜体_、~~删除~~）
- 链接（[上中初二 Wiki](https://zack1219happy.github.io/shsg8c1wiki)）
- 图片
![测试](_assets/test.jpg "测试图")
- 无序列表：
  - a
    * b
      + c
- 有序列表：
  1. abc
  2. ABC
- 引用块
> 你看我容易吗？
- 行内、行间代码块：`code`
```python
print("Hello World!")
```
- 表格：
| a | b |
| - | - |
| c | d |
- 水平线：
---
- LaTeX：$1+1=2$

## 当前公告栏与页面正文支持的语法对比

| 语法功能 | 页面正文（wiki-body） | 公告栏（sidebar-announcement） |
|----------|----------------------|-------------------------------|
| **标题1-6级** | ✅ 完整（`#` ~ `######`），生成对应 `<h1>`~`<h6>`，并被右侧目录捕获 | ✅ **仅支持1-3级**：`#` → `<h1>`（自定义样式）、`##` → `<strong>`、`###` → `<em>` |
| **加粗** | ✅ `**text**` 或 `__text__` | ✅ `**text**` 或 `__text__` |
| **斜体** | ✅ `*text*` 或 `_text_` | ✅ `*text*` 或 `_text_` |
| **删除线** | ✅ `~~text~~` | ✅ `~~text~~` |
| **链接** | ✅ `[text](url)` | ✅ `[text](url)`（新标签页打开） |
| **图片** | ✅ `![alt](url)`，自动添加图注，点击打开原图 | ❌ 不支持（会显示原始文本） |
| **无序列表** | ✅ 支持嵌套（`-`、`*`、`+`） | ✅ 仅支持顶层 `-` 或 `*`（渲染为 `•` 文本，非真实列表） |
| **有序列表** | ✅ `1. item` | ❌ 不支持 |
| **引用块** | ✅ `> quote` | ❌ 不支持 |
| **代码块 / 行内代码** | ✅ ` ``` ` 或 `` `code` `` | ❌ 不支持 |
| **表格** | ✅ 标准 Markdown 表格 | ❌ 不支持 |
| **水平线** | ✅ `---` 或 `***` | ✅ `---` |
| **LaTeX 数学公式** | ✅ 行内 `$...$`，块级 `$$...$$`（通过 KaTeX 渲染） | ✅ 支持（需在公告栏代码中调用 `renderMathInElement`，已实现） |
| **转义字符** | ✅ `\*` 等 | ❌ 不支持 |
